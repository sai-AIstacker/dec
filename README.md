# Decan SIS - Version 1.0

**Decan** is a modern, premium Student Information System designed for schools that demand clarity, efficiency, and a state-of-the-art user experience. 

Powered by a robust backend and a revitalized, fluid interface, Decan 1.0 represents the new standard in school management software.

## Key Features
- **Student Management**: Full lifecycle tracking from enrollment to graduation.
- **Academic Tools**: Advanced gradebooks, scheduling, and attendance modules.
- **Modern UI**: A sleek, geometric-inspired interface with glassmorphism and fluid animations.
- **Extensible**: Support for modules and plugins to adapt to any school's needs.
- **Secure**: Built with modern security practices and data integrity at its core.
